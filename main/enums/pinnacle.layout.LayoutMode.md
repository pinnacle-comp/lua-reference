# Enum `pinnacle.layout.LayoutMode`
<Badge type="tip" text="key" />

`"tiled"` | `"floating"` | `"fullscreen"` | `"maximized"`

A window's current layout mode.

## Values

### `"tiled"`

The window is tiled.

### `"floating"`

The window is floating.

### `"fullscreen"`

The window is fullscreen.

### `"maximized"`

The window is maximized.

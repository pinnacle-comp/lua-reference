# Class `pinnacle.layout.builtin.Floating` : <code><a href="/lua-reference/0.2.0-alpha.1/classes/pinnacle.layout.LayoutGenerator">pinnacle.layout.LayoutGenerator</a></code>


A layout generator that floats windows.

This works by simply returning an empty layout tree.
Note: the windows are not truly floating, see `WindowHandle::spilled` for
details.
